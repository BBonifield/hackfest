API Mashup Smashup

Quick Left Hackfest June 2011

Welcome.

This month we are focusing on public API integration/mashup/utility/debauchery.
Services that group various services' APIs like Mashery, 3scale, and Apigee are becoming more and more prevalent and automatically lend themselves to API mashup goodness-- that is squashing up a couple different services via their API to make a (better?) standalone product.

So, leveraging one of the above company's tools (have you seen Apigee's console?!), or your own knowledge of independent APIs, let the smashing good time begin!

The Rules:

  • Use a least 2 services
  • 3 gets you bonus points
  • Language/platform agnostic
  • Team size: 1-4
  • Pairs strongly encouraged
  • No paid access APIs
  • No scrapers

Process:

  1. Fork
  2. Make
  3. Pull request

Need help? Ask any Quick Lefter.

Judgement:

Judged entirely on radness.

Breaking away from the usual twitter mashup will put you ahead of the competition, we suggest looking into alternative data sources [like Sunlight's Congress API (with JSONP goodness)].

Your pair/team/self will be given 5 minutes to present (our platform or yours), in the order the pull requests were received, in front of our entirely unbiased and completely qualified panel.

Hard code stop is at 9pm, so get to nomming the JSON (or XML...)!
